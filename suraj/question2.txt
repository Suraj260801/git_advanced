1.the command for setting multiple remote repositories is - git remmote add remote-id remote-url
2.Define a git remote which will point to multiple git remotes.
3.Say, we call it “all”: git remote add all REMOTE-URL-1.
4.Register 1st push URL: git remote set-url --add --push all REMOTE-URL-1.
5.Register 2nd push URL: git remote set-url --add --push all REMOTE-URL-2.
6.Push a branch to all the remotes with git push all BRANCH – replace BRANCH with a real branch name.
7.You cannot pull from multiple remotes, but you can fetch updates from multiple remotes with git fetch --all.
